About This Platform

What is StatPulse?

StatPulse is an automated SDMX API monitoring and intelligence platform built for the OECD statistical data ecosystem. It monitors production OECD endpoints and the SIS-CC .Stat Suite demo instance every 6 hours, detects anomalies, tracks catalogue changes, and publishes weekly governance reports automatically.

SDMX Standard

SDMX (Statistical Data and Metadata eXchange) is the ISO 17369 international standard for sharing official statistics. Every OECD dataset on this platform is accessible through the SDMX REST API — the same standard used by Eurostat, IMF, World Bank, and 150+ national statistics offices.

Coverage

Composite Leading Indicators (CLI) · Monthly Unemployment Rates (LFS) · GDP National Accounts · Health Expenditure (SHA) · Full Dataflow Catalogue · SIS-CC Structures, Data Query, and Codelists. Updated every 6 hours via GitHub Actions.
